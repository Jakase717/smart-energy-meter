Project Overview

This project is a smart energy meter designed using ESP32 to monitor voltage, current, and power in real-time. It also includes over/under-voltage protection using a relay and displays data on a web dashboard.

Components Used

  • ESP32 (Microcontroller) – Central unit (Input + Output)
  • ZMPT101B (Voltage Sensor) – AC voltage input
  • ACS712 (Current Sensor) – AC current input
  • Relay Module – Controls load based on voltage thresholds (Output)
  • USB Power Bank – Power supply
  • Resistors & Capacitors – Signal conditioning
  • Buck Converter – To step down voltage to safe ESP32 levels
  • Web Dashboard – Output display (via Wi-Fi)
